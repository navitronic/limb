## Getting Started

- Edit `_config.yml` to set your site title and URL
- Add posts to `_posts/` using the `YYYY-MM-DD-title.md` naming convention
- Customize layouts in `_layouts/` and includes in `_includes/`
- Add static assets to `assets/`
